# 缠论策略说明文档

## 📊 策略对比

我创建了两个缠论策略示例，展示了策略优化的过程：

### 策略1：简单笔策略 (`simple_bi_strategy.py`)

**策略逻辑：**
- 当出现向上笔且力度较强时，做多
- 当出现向下笔且力度较强时，做空
- 使用固定止损（5%）和止盈（10%）

**回测结果：**
- 总交易次数：7次
- 胜率：0%（全部亏损）
- 总收益率：-46.03%
- 平均每笔亏损：-657.57元

**问题分析：**
1. 信号过于简单，没有考虑市场环境
2. 固定止损容易被震荡行情扫损
3. 缺乏对趋势和背驰的判断

---

### 策略2：改进笔策略 (`improved_bi_strategy.py`) ✅ 推荐

**策略逻辑：**
1. **背驰信号**：当价格创新高/新低但力度减弱时，反向开仓（逆势抄底/摸顶）
2. **趋势跟随**：当连续出现同方向的强势笔时，顺势开仓
3. **动态止损**：使用前一笔的极值作为止损位（更符合缠论原理）
4. **固定止盈**：达到8%收益时止盈

**回测结果：**
- 总交易次数：17次
- 胜率：52.94%（9胜8负）
- 总收益率：2.38%
- 平均盈利：196.86元
- 平均亏损：-191.68元
- 盈亏比：约1:1
- 最大回撤：808.38元

**优势：**
1. 结合了背驰和趋势两种信号，更全面
2. 动态止损更符合缠论的笔结构
3. 胜率和盈亏比相对均衡

## 🎯 核心概念解释

### 1. 笔（Bi）
缠论中的基本分析单位，由相邻的顶分型和底分型构成。

```
向上笔：底分型 → 顶分型
向下笔：顶分型 → 底分型
```

### 2. 笔的力度
衡量笔的强弱程度：
```python
力度 = (价格变化幅度 / 起点价格) / 时间跨度
```

### 3. 背驰
价格创新高/新低，但力度减弱，预示趋势可能反转。

**判断标准：**
- 向上笔背驰：新高 + 力度减弱30%以上 → 做空信号
- 向下笔背驰：新低 + 力度减弱30%以上 → 做多信号

### 4. 趋势跟随
当连续出现同方向的强势笔时，顺势而为。

**判断标准：**
- 最近4笔中有3笔同方向
- 最新笔的力度 > 0.5

## 📈 使用方法

### 运行简单策略
```bash
source .venv/bin/activate
python examples/simple_bi_strategy.py
```

### 运行改进策略
```bash
source .venv/bin/activate
python examples/improved_bi_strategy.py
```

### 修改参数
可以在代码中调整以下参数：

```python
strategy = ImprovedBiStrategy(
    symbol="000001.XSHE",
    min_bi_count=5,         # 最少笔数量（建议5-10）
    take_profit_pct=0.08,   # 止盈比例（建议0.05-0.15）
)
```

### 更换股票
修改 `get_kline` 中的 symbol 参数：

```python
bars = get_kline(
    symbol="600000.XSHG",  # 改为其他股票代码
    freq="D",
    start_date="2023-01-01",
    end_date="2024-12-31",
)
```

## 🔧 策略优化建议

### 1. 增加过滤条件
- 成交量确认：放量突破更可靠
- 均线过滤：只在均线多头排列时做多
- 波动率过滤：避免在低波动期交易

### 2. 优化止损止盈
- 使用ATR（平均真实波幅）动态调整
- 移动止损：盈利后逐步提高止损位
- 分批止盈：部分仓位先止盈，剩余仓位追求更大收益

### 3. 仓位管理
- 根据信号强度调整仓位
- 连续亏损后降低仓位
- 盈利后适当加仓

### 4. 多周期结合
```python
# 日线判断趋势，30分钟找入场点
bars_daily = get_kline(symbol="000001.XSHE", freq="D", ...)
bars_30min = get_kline(symbol="000001.XSHE", freq="30min", ...)
```

## ⚠️ 风险提示

1. **回测≠实盘**：历史数据回测的结果不代表未来表现
2. **滑点和手续费**：实盘交易需要考虑交易成本
3. **过度拟合**：避免针对特定股票过度优化参数
4. **资金管理**：永远不要满仓，控制单笔风险
5. **心态管理**：严格执行策略，不要情绪化交易

## 💡 总结

这两个策略展示了：
1. 简单的技术指标策略往往效果不佳
2. 结合缠论的核心思想（背驰、趋势）可以提升效果
3. 动态止损比固定止损更符合市场规律
4. 策略需要不断测试和优化
