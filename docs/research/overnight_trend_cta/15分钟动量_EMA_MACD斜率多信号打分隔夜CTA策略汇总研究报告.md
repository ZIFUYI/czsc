# 15分钟动量 / EMA / MACD / 斜率多信号打分隔夜 CTA 策略汇总研究报告

报告日期：2026-05-22

## 1. 研究目标与结论

本研究围绕 15 分钟 K 线开发可隔夜持仓的权益指数 CTA 策略，覆盖主标的 `000852.XSHG`，并扩展验证 `000905.XSHG` 与 `159915.XSHE`。研究目标为：

- 统计区间：2020-01-01 至 2026-05-20；
- K 线周期：15 分钟，不使用更小周期；
- 策略定位：小时级别到日度级别趋势跟踪，可隔夜；
- 目标指标：全样本年化收益率 >20%，收益回撤比 >3；
- 风控偏好：尽量将最大回撤控制在 10% 附近或以内。

主要结论：

1. `000852.XSHG` 的二阶段优化候选明显优于初版四腿动量策略。候选 B `BAL_VOTE3_S100_V34_1987472d` 全样本年化 43.24%，最大回撤 -9.55%；候选 A `BAL_AVG4_S90_c9bd249e` 更均衡，年化 34.00%，最大回撤 -7.77%。
2. `000852.XSHG` 的参数不能直接迁移到 `000905.XSHG` 和 `159915.XSHE`。直接迁移后，两者全样本年化均约 16%，未达到 20% 目标。
3. 对 `000905.XSHG` 和 `159915.XSHE` 分别做参数优化后，均能找到年化 >20%、收益回撤比 >3 的候选；但两者都没有找到最大回撤低于 10% 的全样本候选。
4. 当前策略的有效性来自趋势信号组合与风险缩放，不宜继续只做无约束参数搜索。下一阶段重点应放在波动率目标、趋势环境过滤、净值回撤降仓、跨品种组合与执行约束验证。

## 2. 策略逻辑

### 2.1 基础思想

策略使用 15 分钟 K 线上的趋势方向信号生成连续权重，权重范围通常为 `[-1, 1]` 或经过风险缩放后的 `[-scale, scale]`。信号只在趋势方向明确反转时改变方向；当指标进入中性区间时，延续上一根 K 线方向，而不是立即清仓。

这个设计适合隔夜趋势跟踪：

- 突破阈值后进入趋势；
- 趋势未反向突破前继续持有；
- 多个信号不一致时通过平均或投票自然降仓；
- 使用上一根 15 分钟 K 线权重计算下一根收益，避免同 K 线前视。

### 2.2 单信号方向生成

每个基础信号先计算一个连续分数 `score_t`，再按阈值生成方向：

```text
weight_t = +1, if score_t > threshold
weight_t = -1, if score_t < -threshold
weight_t = previous_weight, otherwise
```

其中 `threshold = B / 10000`，`B` 的单位为 BP。例如 `B=10` 表示 0.10%。

### 2.3 多信号组合方式

本研究主要使用两类组合。

平均型组合：

```text
final_weight_t = scale * mean(signal_1_t, signal_2_t, ..., signal_k_t)
```

该方式保留信号分歧带来的自然降仓。例如 4 个信号中 3 个看多、1 个看空，平均方向为 `0.5`，若 `scale=0.9`，最终权重为 `+0.45`。

投票型组合：

```text
avg_signal_t = mean(signal_1_t, signal_2_t, ..., signal_k_t)
final_weight_t = +scale, if avg_signal_t >= vote_threshold
final_weight_t = -scale, if avg_signal_t <= -vote_threshold
final_weight_t = 0, otherwise
```

投票型更强调信号一致性，持仓更集中；平均型更平滑，适合降低单次切换冲击。

## 3. 信号含义

### 3.1 信号命名规则

| 信号形式 | 含义 | 计算方式 | 方向规则 |
| --- | --- | --- | --- |
| `MOM_N{N}_B{B}` | 时间序列动量 | `close_t / close_{t-N} - 1` | 动量高于 `B` BP 做多，低于 `-B` BP 做空 |
| `EMA_F{F}_S{S}_B{B}` | EMA 均线趋势 | `EMA(close, F) / EMA(close, S) - 1` | 快 EMA 强于慢 EMA 超阈值做多，反之做空 |
| `MACD_F{F}_S{S}_B{B}` | MACD 式 EMA 价差趋势 | `(EMA(close, F) - EMA(close, S)) / close` | 价差占价格比例超阈值做多或做空 |
| `ESLOPE_N{N}_L{L}_B{B}` | EMA 斜率趋势 | `EMA(close, N)_t / EMA(close, N)_{t-L} - 1` | EMA 斜率超阈值做多或做空 |
| `DON_N{N}` | Donchian 突破 | 突破过去 N 根最高价 / 最低价 | 向上突破做多，向下突破做空 |

### 3.2 组合命名规则

| 组合形式 | 含义 |
| --- | --- |
| `AVG{k}_S{scale}` | k 个信号平均后乘风险缩放，`S90` 表示 `scale=0.90` |
| `VOTE{k}_S{scale}_V{threshold}` | k 个信号投票，`S100` 表示满仓缩放 1.00，`V34` 表示投票阈值约 0.34 |

## 4. 策略版本

### 4.1 初版四腿动量策略

策略名：`ENS4_N32B30_N48B40_N64B10_N64B70_S80`

```text
final_weight = 0.8 * mean(
    MOM_N32_B30,
    MOM_N48_B40,
    MOM_N64_B10,
    MOM_N64_B70,
)
```

该版本用于验证 15 分钟动量隔夜持仓的可行性。它将单腿 `MOM_N64_B20` 的最大回撤从 -17.94% 降至 -9.14%，但 2025 年以来收益偏弱。

### 4.2 000852 二阶段优化候选

候选 A，平衡增强型：`BAL_AVG4_S90_c9bd249e`

```text
weight = 0.9 * mean(
    MOM_N16_B10,
    MOM_N64_B10,
    MACD_F12_S26_B10,
    EMA_F12_S32_B5,
)
```

候选 B，高收益投票型：`BAL_VOTE3_S100_V34_1987472d`

```text
avg_signal = mean(EMA_F8_S32_B0, MOM_N16_B10, MOM_N64_B10)
weight = +1.0 if avg_signal >= 0.34
weight = -1.0 if avg_signal <= -0.34
weight = 0.0 otherwise
```

候选 C，近期稳健投票型：`BAL_VOTE4_S85_V67_483076a0`

```text
avg_signal = mean(MOM_N16_B10, MOM_N48_B40, MOM_N64_B0, MACD_F12_S26_B10)
weight = +0.85 if avg_signal >= 0.67
weight = -0.85 if avg_signal <= -0.67
weight = 0.0 otherwise
```

### 4.3 000905 单标的优化候选

推荐候选：`VOTE4_S90_V67_7c763b5c`

```text
avg_signal = mean(
    ESLOPE_N96_L8_B0,
    EMA_F8_S96_B0,
    ESLOPE_N32_L8_B30,
    EMA_F16_S48_B30,
)
weight = +0.90 if avg_signal >= 0.67
weight = -0.90 if avg_signal <= -0.67
```

`000905.XSHG` 更适合中长 EMA 趋势与 EMA 斜率组合。直接使用 `000852.XSHG` 的参数，全样本年化不足 20%；单独优化后可以达到目标，但最大回撤仍略高于 10%。

### 4.4 159915 单标的优化候选

推荐候选：`AVG3_S60_3de13650`

```text
weight = 0.6 * mean(
    MOM_N56_B100,
    MOM_N56_B50,
    MOM_N56_B70,
)
```

`159915.XSHE` 对参数更敏感。最有效信号集中在约 56 根 15 分钟 K 线的动量尺度上，并且需要降低整体风险缩放到 0.60。需要注意，`159915.XSHE` 是 ETF，本报告中的多空结果属于研究口径；实盘空头侧需要用可交易替代工具实现。

## 5. 回测方式

### 5.1 数据与样本

- 数据源：JQData 15 分钟 K 线；
- 指标预热：2018-01-01 起；
- 统计区间：2020-01-01 至 2026-05-20；
- 标的：`000852.XSHG`、`000905.XSHG`、`159915.XSHE`；
- 收益频率：15 分钟收益按权重计算后聚合到日收益；
- 交易成本：权重变化按 `fee_rate = 0.0002` 扣除；
- 信号执行：使用上一根 15 分钟 K 线权重计算当前 K 线收益，避免前视。

### 5.2 评价指标

| 指标 | 说明 |
| --- | --- |
| 年化收益率 | 基于日收益序列年化 |
| 期末净值 | 初始净值 1.0 的累计结果 |
| 最大回撤 | 日度净值口径最大回撤 |
| Calmar | 年化收益率 / 最大回撤绝对值 |
| 收益回撤比 | 累计收益 / 最大回撤绝对值 |
| 日胜率 | 日收益大于 0 的比例 |
| 换手 | 15 分钟权重变化绝对值之和 |
| 活跃比例 | 非零权重 K 线占比 |

## 8. 风险与局限

1. `000852.XSHG` 二阶段结果很好，但仍可能存在样本内优化风险，需要滚动样本外验证。
2. `000905.XSHG` 和 `159915.XSHE` 单独优化后收益达标，但最大回撤没有压到 10% 以内。
3. 当前回测是指数 / ETF 研究口径，多空执行需要结合实际工具。指数本身不可直接交易，ETF 空头也有执行限制。
4. 交易成本只使用权重变化 2BP 的简化模型，尚未细分滑点、冲击成本、期货展期、保证金占用等真实约束。
5. 信号主要仍是价格趋势类信号，缺少波动率、成交量、市场宽度、风格环境、CZSC 高周期结构等过滤条件。

## 9. 后续研究与优化方向

### 9.1 风险压缩优先

`000905.XSHG` 和 `159915.XSHE` 的主要问题不是收益不足，而是回撤偏高。下一阶段应优先测试：

- 波动率目标：高波动阶段自动降低 `scale`；
- 净值回撤降仓：策略净值进入回撤区间时降低风险预算；
- 单笔亏损控制：连续不利波动时减仓或冷却；
- 权重变化约束：减少震荡阶段频繁换向。

### 9.2 趋势环境过滤

2025 年以来趋势质量下降，单纯扩大参数网格容易过拟合。更合理的方向是增加环境过滤：

- 60 分钟 / 日线趋势强度过滤；
- 长周期均线斜率过滤；
- 高周期 CZSC 笔、线段、中枢状态过滤；
- 低波动率或低趋势强度阶段降权；
- 市场风格切换过滤，例如大小盘相对强弱。

### 9.3 多标的组合

单标的优化容易暴露标的风格风险。可以将 `000852`、`000905`、`159915` 的策略权重组合起来，评估组合层面：

- 收益与回撤是否互补；
- 多标的信号相关性；
- 同向暴露是否过高；
- 组合波动率目标能否把最大回撤压到 10% 以内；
- 是否需要分标的参数，而不是统一参数。

### 9.4 参数稳定性与样本外检验

需要补充：

- 参数邻域稳定性图；
- 滚动训练 / 验证切片；
- 2020-2022、2023-2024、2025-2026 分段样本外；
- 成本从 1BP 到 5BP 的压力测试；
- 候选 A/B/C 与单标的优化候选的换手和持仓周期对比。

### 9.5 生产化改造

如果进入正式策略库，需要将研究脚本整理为可配置策略：

- 参数、信号组合、风险缩放、费率配置化；
- 输出标准化权重文件、交易段统计、分年评估；
- 增加固定数据切片测试；
- 明确实盘执行标的和空头替代工具；
- 将研究级回测与真实交易价格模型分开。

## 10. 最终建议

当前建议分三层使用：

1. `000852.XSHG`：优先保留候选 A 和候选 C 作为稳健版本，候选 B 作为高收益版本继续压力测试。
2. `000905.XSHG`：不直接迁移 `000852` 参数，使用 `VOTE4_S90_V67_7c763b5c` 作为后续研究基线。
3. `159915.XSHE`：使用 `AVG3_S60_3de13650` 作为研究基线，但必须继续做回撤压缩和空头执行可行性验证。

总体上，隔夜趋势 CTA 方向是有效的，但当前还处于研究阶段。`000852.XSHG` 已经形成较强候选；`000905.XSHG` 和 `159915.XSHE` 证明了“分标的参数优化”的必要性。下一步不应继续只追求更高年化，而应围绕回撤压缩、跨标的组合和样本外稳定性推进。
