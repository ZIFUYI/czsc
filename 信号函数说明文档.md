## 信号函数命名规则

CZSC 信号函数遵循统一的命名规则：

```
{模块前缀}_{信号类型}_V{版本日期}
```

- **模块前缀**：
  - `cxt` - 缠论形态（CZSC形态）
  - `bar` - K线基础
  - `tas` - 技术指标（TA-Lib Signals）
  - `vol` - 成交量
  - `jcc` - 日本蜡烛图
  - `pos` - 持仓管理
  - `byi` - 白仪贡献
  - `coo` - 组合信号
  - `ang` - 琅盎贡献
  - `zdy` - 自定义
  - `xls` - 谢磊贡献

- **版本日期**：YYMMDD 格式，如 V230228 表示 2023年2月28日

## 信号函数使用示例

```python
from czsc import CZSC
from czsc.signals.cxt import cxt_bi_status_V230101
from czsc.signals.tas import tas_macd_base_V221028

# 创建 CZSC 对象
czsc = CZSC(bars)

# 调用信号函数
signal1 = cxt_bi_status_V230101(czsc, di=1)
signal2 = tas_macd_base_V221028(czsc, di=1)

print(signal1)  # OrderedDict 格式的信号结果
print(signal2)
```
